objective/problem

objective:

  • modifying & creating images
  • change colors within an image

problem:

  • images need to be modified

technical implementation

  • our solution uses a library for python called 'pillow'
  • allows us to open, manipulate, save many different image formats
  • PIL = python image library
  • pillow builds ontop of PIL, PIL is deprecated

computing techniques

  • .convert - changes the mode of an image, for example, RGB to Luminance
  • .save - stores the image with the modified pixel values and correct filetype header
  • .invert - takes the value of each color and subtracts the original color value from 255 and saves the result
  • .solarize - inverts all colors above a certain threshold value
